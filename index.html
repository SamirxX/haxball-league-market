Player & Team Commands

!register [Haxball_Nick]
Registers a user into the league database and assigns the @Free Agent role.
!createteam [Team_Name] [Tag]
Creates a new team entry, creates team role & captain permissions.
!sign @Player
Captains sign a free agent to their team roster.
!release @Player
Releases a player back to the free agents market.

Match & League Commands

!report @Home [score] - [score] @Away
Submits match score, updates league table, and records goal diff.
!standings
Outputs the live league standings table directly into Discord chat.
!roster @Team
Displays current active roster and squad list for a team.
!freeagents
Lists all available free agents ready for team contracts.

Step-by-Step Server Creation Guide

1

Create Categories & Roles

Set up the exact category and role structure provided in the Visualizer tab. Assign color hex codes to differentiate League Admins, Captains, and Teams.

2

Configure Channel Permissions

Restrict #announcements and #standings to read-only for regular players. Give Captains permission to post in #match-reporting and #transfers.

3

Deploy the Haxball League Bot

Install discord.py on your computer or VPS, insert your Bot Token, and run python bot.py to handle automatic standings and match reports.

4

Set Up Haxball Headless Room (Optional)

Integrate a Haxball Headless Host script to automatically broadcast room links to #haxball-links whenever matches are ready to be played!
